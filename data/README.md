# Data

**What's in this folder:**
- `raw_logs/` - Original Windows log files (.evtx format)
- `processed/` - Converted and cleaned log files
- `features/` - Important data extracted from logs
- `intermediate/` - Temporary files during processing
- `detection_results/` - Results from security analysis
- `uploads/` - Files uploaded from external sources
- `extracted/` - Files extracted from uploads

**Purpose:** This folder contains all the Windows log data at every stage of processing - from the original raw files to the final analysis results.

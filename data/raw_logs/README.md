# Raw Logs

**What's in this folder:**
- `Application.evtx` - Windows Application event logs
- `PowerShell_Operational.evtx` - PowerShell activity logs
- `Security.evtx` - Windows Security event logs
- `System.evtx` - Windows System event logs

**Purpose:** This folder contains the original Windows log files in their native .evtx format. These are the raw files directly from Windows computers before any processing.
